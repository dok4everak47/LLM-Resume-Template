    \item[\faCode] \textbf{MetroType China — 数据驱动的地铁主题打字应用}

    以中国大陆四城（北京 / 上海 / 广州 / 深圳）25 条线路、684 座车站的中英文站名为题的打字练习网站；城市与线路存于单一 \texttt{metro.json}，新增城市 / 线路无需改动应用逻辑。技术栈: React 18, Vite 5, pnpm, d3-geo, topojson-client, Python
    \begin{itemize}
        \item 用 Python 脚本从公开资料生成结构化站点 / 线路数据（经纬度、中英文站名、拼音），覆盖环线（北京 2 / 10 号、上海 4 号）与支线的真实服务区段

        \item 两级导航（城市概览 → 线路详情 → 打字）、深色模式、全键盘操作、响应式与 \texttt{prefers-reduced-motion} 无障碍适配
